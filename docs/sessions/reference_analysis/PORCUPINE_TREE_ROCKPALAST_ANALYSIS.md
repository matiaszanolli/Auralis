Analyzing Porcupine Tree's 2006 Rockpalast live concert provides **critical insights into live recording mastering** that differ substantially from studio masters (Deep Purple).

This is **NOT** a case of "over-boosting bass" - this is a case of **correcting bootleg characteristics**!

## üîç Key Discoveries

### Discovery 1: Bootleg Recordings Are EXTREMELY Dark

**Evidence**:
- Spectral centroid: 374-571 Hz (vs. normal 650-1000 Hz)
- Bass-to-mid: +13.6 to +16.8 dB (vs. normal +1-2 dB)
- Treble-to-mid: -17.8 dB (basically absent)

**Why This Happens**:
- Live recording captures room acoustics poorly
- Speaker placement/angle affects frequency response
- Bootleg recording equipment has poor treble
- Bass frequencies travel better in venues

**Implication**: **Bootleg ‚â† Bad mastering. It's a recording characteristic.**

---

### Discovery 2: Bootleg Stereo Is Mono-ish

**Evidence**:
- Width: 0.17-0.23 (vs. professional 0.4-0.6)
- Correlation: 0.77-0.83 (vs. professional 0.5-0.7)

**Why This Happens**:
- Bootleg recording position captures narrow stereo image
- Can be from single microphone or pair placed suboptimally
- Live recording angle affects stereo capture

**Implication**: **Stereo expansion is valid mastering technique for bootlegs**

---

### Discovery 3: Matchering's Mastering Philosophy for Live Recordings

**Approach**:
1. **Aggressive brightening** (+439 to +599 Hz spectral shift)
2. **Bass reduction** (more aggressive than studio -2.3 to -5.5 dB)
3. **Stereo expansion** (100% wider)
4. **Loudness increase** (+1.1 to +3.2 dB) while maintaining peaks
5. **Dynamic range preservation** (consistent +23.5 dB expansion)

**Rationale**:
- Bootleg is too dark ‚Üí brighten aggressively
- Bootleg is too bass-heavy ‚Üí reduce bass significantly
- Bootleg is too narrow ‚Üí expand stereo
- Can increase overall loudness because original peaks are already hot
- Expand DR because bootleg is compressed from recording process

### Key Insight: Studio vs Live Processing Are Different

**Deep Purple (Studio Live - Professional Mix)**:
- Bass: +1.15 dB (modest)
- Treble: +2.3 dB (modest)
- Stereo: minimal change (already good)

**Porcupine Tree (Live Bootleg - Raw Recording)**:
- Bass reduction: -3 to -5.5 dB (aggressive!)
- Treble boost: +2 to +4.6 dB (moderate)
- Stereo expansion: +0.21-0.23 (major)
- DR expansion: +23.5 dB (major)
- RMS increase: +1.1 to +3.2 dB (substantial)

### Recommendation 1: Detect Recording Type

**Bootleg characteristics to identify**:
- Spectral centroid < 500 Hz (very dark)
- Bass-to-mid ratio > +12 dB (excessive bass)
- Stereo width < 0.3 (narrow/mono)
- Usually presents all three together

**Different Processing for**:
- Professional studio recordings (like Deep Purple Steven Wilson)
- Live concert recordings (like Porcupine Tree)
- Bootleg recordings (like Porcupine Tree Rockpalast)

### Recommendation 2: Aggressive Treble Boost for Bootlegs

**Finding**: Bootlegs need **much more treble** than studio recordings

**Steven Wilson studio**: +2.3 dB treble boost
**Matchering bootleg**: +2.4 to +4.6 dB treble boost

**Auralis Action**: For detected bootlegs, increase treble boost aggressiveness

### Recommendation 3: Bass Reduction Not Bass Boost

**Finding**: Even though bass seems boosted, it's actually **reduction from excessive bootleg bass**

**Pattern**:
- Bootleg: +16 dB bass-to-mid (way too much)
- Professional: +10-12 dB bass-to-mid (reduced toward normal)
- Normal music: +1-2 dB bass-to-mid

**Auralis Action**: For bootlegs, apply BASS REDUCTION (not boost) to bring bass down to normal levels

### Recommendation 4: Stereo Expansion is Valid

**Finding**: Expanding narrow bootleg stereo image is standard technique

**Matchering approach**:
- Bootleg width 0.17-0.23 ‚Üí Professional width 0.38-0.46 (100% expansion)
- Reduces L/R correlation to create width perception

**Auralis Action**: Implement optional stereo expansion for narrow recordings
