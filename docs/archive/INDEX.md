# Historical Documentation Archive

This directory contains 63 historical documentation files from previous phases of Auralis development. These are kept for reference but are no longer actively used.

## Organization

- **Phase Documentation**: PHASE_*.md - Completed development phases (0-11)
- **Design Documents**: DESIGN_*.md, AURALIS_*.md - Previous design iterations
- **Session Notes**: SESSION_*.md, CONVERSATION_*.md - Past session summaries
- **Completed Features**: FRONTEND_*.md, RUST_*.md, GPU_*.md, etc. - Completed feature documentation
- **Archived Plans**: MIGRATION_*.md, DSP_*.md, etc. - Executed plans and migrations

## Current Documentation Structure

See the root directory for current documentation:
- **README.md** - Main project overview
- **CLAUDE.md** - Instructions for Claude Code development
- **START_HERE.md** - Quick start guide
- **LAUNCH.md** - How to launch the application
- **CHANGELOG.md** - Version history

For development documentation, see:
- **docs/README.md** - Documentation index
- **DEVELOPMENT_SETUP_BACKEND.md** - Backend setup guide
- **DEVELOPMENT_SETUP_FRONTEND.md** - Frontend setup guide
